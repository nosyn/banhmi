# @banhmi/s3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the `@banhmi/s3` package that wraps Bun's native `S3Client` with a NestJS-style module, injectable token, and helper methods for common S3 operations.

**Architecture:** `S3Module.forRoot(config)` creates an inner module class (same pattern as SqliteModule) providing an `S3Client` instance. `@InjectS3()` is a marker decorator; actual injection uses `S3_TOKEN` via `static inject`. Helper methods (`upload`, `download`, `presign`, `delete`) are instance methods on an injectable `S3Service` that wraps the client. Bun 1.1+ ships a native `S3Client` — no AWS SDK needed.

**Tech Stack:** `bun` native `S3Client` (zero deps), TC39 Stage 3 decorators, Bun test runner

import { Module } from '@banhmi/common'
import type { S3Client } from 'bun'
import { S3_TOKEN } from './tokens'

export interface S3ModuleConfig {
  bucket?: string
  region?: string
  endpoint?: string
  accessKeyId?: string
  secretAccessKey?: string
  sessionToken?: string
}

export class S3Module {
  static forRoot(config: S3ModuleConfig) {
    @Module({
      providers: [
        {
          provide: S3_TOKEN,
          useFactory: async (): Promise<S3Client> => {
            const { S3Client: BunS3Client } = await import('bun')
            return new BunS3Client(config)
          },
        },
      ],
      exports: [S3_TOKEN],
    })
    class S3RootModule {}

    return S3RootModule
  }
}
```

- [ ] **Step 3: Create `packages/s3/src/s3.service.ts`**

import { Injectable } from '@banhmi/common'
import type { S3Client } from 'bun'
import { S3_TOKEN } from './tokens'

@Injectable()
export class S3Service {
  static inject = [S3_TOKEN] as const

  constructor(private readonly client: S3Client) {}

  async upload(key: string, data: string | ArrayBuffer | Uint8Array | Blob): Promise<number> {
    const file = this.client.file(key)
    return file.write(data)
  }

  async download(key: string): Promise<Blob> {
    const file = this.client.file(key)
    return file.arrayBuffer().then((buf) => new Blob([buf]))
  }

  presign(key: string, expiresIn = 3600): string {
    const file = this.client.file(key)
    return file.presign({ expiresIn })
  }

  async delete(key: string): Promise<void> {
    const file = this.client.file(key)
    await file.delete()
  }
}
```
