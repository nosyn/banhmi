# Passport-style auth (`@banhmi/auth`)

`@banhmi/auth` provides strategy-based (Passport-style) authentication for
Banhmi applications. Compose strategies onto routes with `@UseAuth` and read
the principal with `getAuthUser`.

> **Looking for full session management, OAuth, and RBAC?**
> See the [better-auth recipe](./better-auth) for `@banhmi/better-auth`.

## Setup

```bash
bun add @banhmi/auth
```

## Strategies

### Local (username + password)

```ts
import { LocalStrategy, UseAuth, getAuthUser } from '@banhmi/auth'
import { verifyPassword } from '@banhmi/crypto'

const local = new LocalStrategy({
  usernameField: 'email',
  validate: async ({ username, password }) => {
    const user = await db.findByEmail(username)
    if (!user) return null
    return (await verifyPassword(password, user.hash)) ? user : null
  },
})
```

Apply to a login handler:

```ts
@Post('/login')
@UseAuth('local', [local])
login(ctx: RouteCtx) {
  return { user: getAuthUser(ctx) }
}
```

### JWT Bearer

```ts
import { JwtStrategy, UseAuth, getAuthUser } from '@banhmi/auth'

const jwt = new JwtStrategy({
  secret: Bun.env.JWT_SECRET ?? '',
})

@Get('/me')
@UseAuth('jwt', [jwt])
me(ctx: RouteCtx) {
  return { user: getAuthUser(ctx) }
}
```

Custom token extractor (cookie, query param, etc.):

```ts
new JwtStrategy({
  secret: '...',
  extract: (req) => new URL(req.url).searchParams.get('token'),
})
```

### Google / GitHub OAuth

```ts
import { GoogleStrategy, GitHubStrategy } from '@banhmi/auth'

const google = new GoogleStrategy({
  clientId: Bun.env.GOOGLE_CLIENT_ID ?? '',
  clientSecret: Bun.env.GOOGLE_CLIENT_SECRET ?? '',
  redirectUri: 'https://example.com/auth/google/callback',
  stateSecret: Bun.env.STATE_SECRET ?? '',
  validate: async (profile) => db.upsertUser(profile),
})

// Redirect to provider
@Get('/auth/google')
async googleLogin() {
  return Response.redirect(await google.getAuthorizationUrl())
}

// Handle callback — @UseAuth reads code+state from query params
@Get('/auth/google/callback')
@UseAuth('google', [google])
googleCallback(ctx: RouteCtx) {
  const user = getAuthUser(ctx)
  return Response.redirect('/?loggedIn=1')
}
```

## `AuthModule` — DI setup

Register strategies once at the module level and inject the strategy list:

```ts
import { AuthModule, AUTH_STRATEGIES } from '@banhmi/auth'

@Module({
  imports: [
    AuthModule.register({ strategies: [local, jwt, google] }),
  ],
})
class AppModule {}
```

The `AUTH_STRATEGIES` token provides the registered strategy array for
programmatic use.

## Custom strategies

Extend `Strategy<TUser>`:

```ts
import { Strategy } from '@banhmi/auth'

class ApiKeyStrategy extends Strategy<{ id: string }> {
  readonly name = 'api-key'

  async authenticate(req: Request) {
    const key = req.headers.get('x-api-key')
    if (!key) return null
    return db.findByApiKey(key)
  }
}
```

## Migrating from Passport.js (NestJS)

If you're migrating from a NestJS + Passport app, Passport strategies can run
as Banhmi middleware during a transition period:

```ts
import passport from 'passport'
import { Strategy as JwtStrategy, ExtractJwt } from 'passport-jwt'

passport.use(new JwtStrategy(
  { jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken(), secretOrKey: 'secret' },
  (payload, done) => done(null, payload),
))

const app = await BanhmiFactory.create(AppModule)
app.use(passport.initialize())
await app.listen(3000)
```
