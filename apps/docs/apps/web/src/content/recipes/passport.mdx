# Passport (auth)

Banhmi ships `@banhmi/auth` as a first-class auth module built on
[Better Auth](https://better-auth.com), which provides sessions, JWT, email OTP,
OAuth providers, and 2FA without requiring Passport.js.

## Recommended: @banhmi/auth + Better Auth

```bash
bun add @banhmi/auth better-auth
```

See the [Authentication](/security/authentication) guide for full setup.

## Using Passport.js (legacy migration path)

If you're migrating from a NestJS + Passport app, Passport strategies run
as Banhmi middleware:

```ts
import passport from 'passport'
import { Strategy as JwtStrategy, ExtractJwt } from 'passport-jwt'

// Configure strategy
passport.use(new JwtStrategy(
  { jwtFromRequest: ExtractJwt.fromAuthHeaderAsBearerToken(), secretOrKey: 'secret' },
  (payload, done) => done(null, payload),
))

// Register as middleware
const app = await BanhmiFactory.create(AppModule)
app.use(passport.initialize())
await app.listen(3000)
```

Then protect routes with a custom guard:

```ts
import type { CanActivate, ExecutionContext } from 'banhmi'
import passport from 'passport'

@Injectable()
export class PassportJwtGuard implements CanActivate {
  canActivate(ctx: ExecutionContext): Promise<boolean> {
    const req = ctx.switchToHttp().getRequest()
    return new Promise((resolve) =>
      passport.authenticate('jwt', { session: false }, (err: unknown, user: unknown) => {
        if (err || !user) resolve(false)
        else { req.user = user; resolve(true) }
      })(req, {}, () => {}),
    )
  }
}
```
