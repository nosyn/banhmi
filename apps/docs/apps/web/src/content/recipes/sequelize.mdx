# Sequelize

Sequelize requires `reflect-metadata` and Node.js native addons that are
incompatible with Bun's module system. Use Drizzle ORM or `@banhmi/postgres`
for Banhmi applications.

## Recommended alternatives

See [TypeORM](/recipes/typeorm) for the full alternatives table.

## Drizzle ORM (recommended)

```bash
bun add @banhmi/drizzle drizzle-orm
```

See [Drizzle ORM](/recipes/drizzle) for setup.

## Direct SQL

```ts
import { PostgresModule } from '@banhmi/postgres'

@Module({
  imports: [PostgresModule.forRoot({ connectionString: Bun.env['DATABASE_URL']! })],
})
export class AppModule {}
```
