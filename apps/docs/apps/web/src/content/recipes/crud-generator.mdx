# CRUD Generator

The `banhmi generate resource` command scaffolds a complete CRUD resource
in one step — controller, service, module, and DTOs with OpenAPI decorators.

## Usage

```bash
banhmi g resource cats
```

Generates:

```
src/cats/
  cats.controller.ts   # GET/POST/GET:id/PATCH:id/DELETE:id
  cats.service.ts      # findAll/findOne/create/update/remove
  cats.module.ts
  dto/
    create-cat.dto.ts  # @ApiProperty decorated input
    update-cat.dto.ts  # PartialType(CreateCatDto)
```

## Transport selection

```bash
banhmi g resource cats --transport http       # default — REST controller
banhmi g resource cats --transport graphql    # GraphQL resolver
banhmi g resource cats --transport microservice  # @MessagePattern handler
banhmi g resource cats --transport websocket  # @SubscribeMessage handler
```

## Skip spec file

```bash
banhmi g resource cats --no-spec
```
