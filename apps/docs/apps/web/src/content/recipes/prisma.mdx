# Prisma

Prisma is compatible with Bun (as of Prisma 5.9+). Wrap the `PrismaClient`
as a Banhmi provider for full DI integration.

## Setup

```bash
bun add prisma @prisma/client
bunx prisma init
```

Create a Prisma provider:

```ts
import { Injectable, OnModuleDestroy, OnModuleInit } from 'banhmi'
import { PrismaClient } from '@prisma/client'

@Injectable()
export class PrismaService extends PrismaClient implements OnModuleInit, OnModuleDestroy {
  async onModuleInit() {
    await this.$connect()
  }

  async onModuleDestroy() {
    await this.$disconnect()
  }
}
```

Register in a module:

```ts
@Module({
  providers: [PrismaService],
  exports: [PrismaService],
})
export class PrismaModule {}
```

Use in a service:

```ts
@Injectable()
export class CatsService {
  static inject = [PrismaService] as const
  constructor(private prisma: PrismaService) {}

  findAll() {
    return this.prisma.cat.findMany()
  }

  create(data: Prisma.CatCreateInput) {
    return this.prisma.cat.create({ data })
  }
}
```
