# TypeORM

TypeORM requires `reflect-metadata` and `emitDecoratorMetadata` which are
incompatible with Banhmi's TC39 Stage 3 decorator model. For Banhmi apps,
use Drizzle ORM or `@banhmi/postgres` instead.

## Recommended alternatives

| Use case | Banhmi module |
|----------|--------------|
| PostgreSQL (type-safe) | [`@banhmi/drizzle`](/recipes/drizzle) |
| PostgreSQL (raw) | [`@banhmi/postgres`](/techniques/database) |
| MySQL | [`@banhmi/mysql`](/techniques/database) |
| SQLite | [`@banhmi/sqlite`](/bun-native/bun-sqlite) |
| MongoDB | [`@banhmi/mongo`](/techniques/mongo) |

## Migrating from NestJS + TypeORM

See the [NestJS Migration Guide](/migration/concepts) for a step-by-step
mapping from TypeORM entities to Drizzle schemas.
