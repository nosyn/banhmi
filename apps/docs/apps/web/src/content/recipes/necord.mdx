# Necord

Necord is a Discord bot framework built on NestJS. A Banhmi-compatible port
is on the roadmap; in the meantime you can run a Necord-based NestJS service
as a microservice that communicates with your Banhmi API.

## Status

Necord requires NestJS core and is not yet compatible with Banhmi's TC39
Stage 3 decorator model. Track progress in the
[Banhmi GitHub issues](https://github.com/banhmi/banhmi/issues).

## Workaround — microservice pattern

Run a NestJS + Necord Discord bot as a separate process and connect it to
your Banhmi API via TCP microservices:

```ts
// In the Necord bot (NestJS)
@Injectable()
export class BotService {
  constructor(@InjectClient() private readonly client: NecordClient) {}

  // Call Banhmi API via HTTP or microservice transport
}
```
