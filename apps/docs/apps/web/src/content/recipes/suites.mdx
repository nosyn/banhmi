# Suites (Automock)

Suites (formerly Automock) is a unit-testing library that automatically
mocks all DI dependencies. It is a third-party library that works with Banhmi.

## Setup

```bash
bun add -d @suites/unit @suites/doubles
```

## Usage

```ts
import { TestBed } from '@suites/unit'
import { CatsService } from './cats.service'
import { CatsRepository } from './cats.repository'

describe('CatsService', () => {
  let service: CatsService
  let repository: jest.Mocked<CatsRepository>

  beforeAll(async () => {
    const { unit, unitRef } = await TestBed.solitary(CatsService).compile()
    service = unit
    repository = unitRef.get(CatsRepository)
  })

  test('findAll returns cats', async () => {
    repository.findAll.mockResolvedValue([{ id: '1', name: 'Tom' }])
    expect(await service.findAll()).toHaveLength(1)
  })
})
```

## Note

Suites is designed for the Jest/Vitest ecosystem. For Bun's test runner,
use `@banhmi/testing` with `useValue` mock providers instead:

```ts
const module = await TestingModule.createTestingModule({
  providers: [
    CatsService,
    { provide: CatsRepository, useValue: { findAll: () => [] } },
  ],
}).compile()
```
