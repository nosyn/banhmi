# Swagger

Banhmi uses `@banhmi/openapi` (renamed from `@banhmi/swagger`) which ships
with Scalar UI instead of the legacy Swagger UI. The package generates a full
OpenAPI 3.1 spec from your controllers and DTOs.

## See the OpenAPI docs

See [OpenAPI Quick Start](/openapi/overview) for full setup, decorator reference,
and Scalar UI configuration.

## Quick reference

```bash
bun add @banhmi/openapi
```

```ts
import { OpenApiModule } from '@banhmi/openapi'

@Module({
  imports: [
    OpenApiModule.forRoot({
      title: 'Cats API',
      version: '1.0',
      scalarUi: true,
    }),
  ],
})
export class AppModule {}
```

Visit `http://localhost:3000/api` for the Scalar UI.
