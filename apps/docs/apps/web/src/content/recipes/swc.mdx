# SWC

SWC is a Rust-based TypeScript compiler used in NestJS to speed up builds.
**Banhmi does not need SWC.** Bun's built-in TypeScript transpiler (also
Rust-based via Zig) is already fast — typically under 1ms per file.

## Why this is not needed in Banhmi

- Bun transpiles TypeScript natively without a separate compiler step.
- `bun run src/main.ts` is direct — no `tsc` or `ts-node` required.
- `bun build` produces optimised bundles for production.

## If you're migrating from NestJS + SWC

Replace:
```json
{ "scripts": { "build": "nest build --webpack" } }
```

With:
```json
{ "scripts": { "build": "banhmi build" } }
```

Type-checking is separate:
```bash
bun run type-check  # runs tsc --noEmit
```
