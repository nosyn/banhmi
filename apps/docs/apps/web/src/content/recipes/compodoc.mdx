# Compodoc

Compodoc is a documentation tool for Angular/NestJS projects. It is not
directly compatible with Banhmi's TC39 Stage 3 decorators, but TSDoc comments
in your code are supported by most TypeScript documentation generators.

## Recommended alternatives

- **OpenAPI / Scalar** — live, interactive API documentation from `@banhmi/openapi`.
- **TypeDoc** — generates static HTML docs from TSDoc comments. Works with Bun.

## TypeDoc setup

```bash
bun add -d typedoc
```

`typedoc.json`:

```json
{
  "entryPoints": ["src/index.ts"],
  "out": "docs-output",
  "exclude": ["**/*.test.ts"]
}
```

Run:

```bash
bunx typedoc
```
