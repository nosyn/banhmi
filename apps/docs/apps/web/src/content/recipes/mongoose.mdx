# Mongoose

`@banhmi/mongo` wraps the official `mongodb` driver as a Banhmi DI module.
For applications that need Mongoose's schema modeling, you can use Mongoose
alongside Banhmi's DI container.

## Recommended: @banhmi/mongo

See the [MongoDB](/techniques/mongo) guide for `@banhmi/mongo` with the
official driver, which is lighter and Bun-compatible.

## Mongoose integration

```bash
bun add mongoose
```

Register as an async provider:

```ts
import mongoose from 'mongoose'
import { Token } from 'banhmi'

export const MONGOOSE_TOKEN = Token<typeof mongoose>('MONGOOSE_TOKEN')

@Module({
  providers: [
    {
      provide: MONGOOSE_TOKEN,
      useFactory: async (config: ConfigService) => {
        await mongoose.connect(config.get('MONGODB_URI'))
        return mongoose
      },
      inject: [ConfigService],
    },
  ],
  exports: [MONGOOSE_TOKEN],
})
export class MongooseModule {}
```

Define a model:

```ts
import { Schema, model } from 'mongoose'

const CatSchema = new Schema({
  name: String,
  age: Number,
})

export const CatModel = model('Cat', CatSchema)
```

Inject the model:

```ts
@Injectable()
export class CatsRepository {
  findAll() {
    return CatModel.find().lean()
  }
}
```
