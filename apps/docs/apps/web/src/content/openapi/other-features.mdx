# Other Features

## SDL export (`generateSdl`)

`generateSdl` walks `@ApiProperty` metadata on model classes and produces a
GraphQL SDL string. This is the bridge between `@banhmi/openapi` (Wave 5) and the
upcoming `@banhmi/graphql` package (Wave 7).

```ts
import { ApiProperty } from '@banhmi/openapi'
import { generateSdl } from '@banhmi/openapi'

class User {
  @ApiProperty({ type: 'number', format: 'int' })
  id: number

  @ApiProperty({ type: 'string' })
  name: string

  @ApiProperty({ type: 'string', required: false })
  bio?: string
}

class Post {
  @ApiProperty({ type: 'string' })
  title: string

  @ApiProperty({ type: User })
  author: User
}

const sdl = generateSdl([User, Post])
// type User {
//   id: Int!
//   name: String!
//   bio: String
// }
//
// type Post {
//   title: String!
//   author: User!
// }
```

### Type mapping

| `@ApiProperty` type    | GraphQL type |
|------------------------|-------------|
| `'string'`             | `String`    |
| `'number'` (default)   | `Float`     |
| `'number'` + `format:'int'` | `Int` |
| `'boolean'`            | `Boolean`   |
| `['string']`           | `[String!]` |
| Class ref              | Type name   |

Required properties (default) end with `!`. Pass `required: false` for optional.

## `@ApiExtraModels`

Register additional model classes in `components/schemas` even if they are not
directly referenced in a route:

```ts
@ApiExtraModels(PaginationDto)
@Controller('/items')
export class ItemsController {}
```
