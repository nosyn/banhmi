# Scalar UI

`@banhmi/openapi` uses [Scalar](https://scalar.com) as the **default** API documentation
renderer. Scalar loads its assets from the jsDelivr CDN. The classic Swagger UI is
available as an opt-in.

## Switching renderers

Pass the `ui` option to `SwaggerModule.setup`:

```ts
// Default — Scalar UI
SwaggerModule.setup('/api', app, doc)
SwaggerModule.setup('/api', app, doc, { ui: 'scalar' })

// Classic Swagger UI
SwaggerModule.setup('/api', app, doc, { ui: 'swagger' })
```

## Scalar theme

Use `renderScalarHtml` directly if you need theme or title customisation:

```ts
import { renderScalarHtml } from '@banhmi/openapi'

const html = renderScalarHtml('/api/openapi.json', {
  theme: 'purple',
  title: 'My API Docs',
})
```

Available themes: `default`, `alternate`, `moon`, `purple`, `solarized`, `bluePlanet`,
`deepSpace`, `saturn`, `kepler`, `elysiajs`, `fastify`, `mars`.

## Serving assets locally

The CDN script tag in `renderScalarHtml` points to:

```
https://cdn.jsdelivr.net/npm/@scalar/api-reference
```

To pin a specific version or serve from a local bundle, call `renderScalarHtml` with
a custom HTML wrapper instead of using `SwaggerModule.setup`.
