# Security

`@banhmi/openapi` provides three security decorators that map to OpenAPI
`securitySchemes`: `@ApiBearerAuth`, `@ApiCookieAuth`, and `@ApiSecurity`.

## Bearer authentication

Register the scheme in `DocumentBuilder`, then apply the decorator:

```ts
// main.ts
const doc = new DocumentBuilder()
  .setTitle('My API')
  .setVersion('1')
  .addBearerAuth()  // registers 'bearerAuth' scheme
  .build()

// controller.ts
@ApiBearerAuth()                    // applies to all routes in controller
@Controller('/admin')
export class AdminController {
  @ApiBearerAuth('custom-scheme')   // or apply per-handler
  @Get('/secret')
  secret() {}
}
```

## Cookie authentication

```ts
const doc = new DocumentBuilder()
  .setTitle('My API')
  .setVersion('1')
  .build()
// Register cookie scheme manually:
// doc.components.securitySchemes.cookieAuth = { type: 'apiKey', in: 'cookie', name: 'session' }

@ApiCookieAuth()
@Controller('/profile')
export class ProfileController {}
```

## Named security schemes

```ts
@ApiSecurity('oauth2-scopes')
@Get('/protected')
protected() {}
```
