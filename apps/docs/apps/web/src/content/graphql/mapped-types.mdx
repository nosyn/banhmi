# Mapped Types

Mapped type helpers derive new input types from existing ones, reducing
boilerplate for CRUD input shapes.

## PartialType

Makes all fields optional (equivalent to TypeScript `Partial<T>`):

```ts
import { InputType } from '@banhmi/graphql'
import { PartialType } from '@banhmi/graphql'
import { CreateCatInput } from './create-cat.input'

@InputType()
export class UpdateCatInput extends PartialType(CreateCatInput) {}
```

## PickType

Selects a subset of fields:

```ts
@InputType()
export class LoginInput extends PickType(UserInput, ['email', 'password'] as const) {}
```

## OmitType

Excludes specific fields:

```ts
@InputType()
export class CreateUserInput extends OmitType(UserInput, ['id'] as const) {}
```

## IntersectionType

Combines two types:

```ts
@InputType()
export class AdminInput extends IntersectionType(CreateUserInput, AdminFields) {}
```

## API reference

- `PartialType(Base)` — all fields become optional.
- `PickType(Base, keys)` — keep only listed keys.
- `OmitType(Base, keys)` — remove listed keys.
- `IntersectionType(A, B)` — merge two input types.
