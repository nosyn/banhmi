# Directives

GraphQL directives annotate types, fields, or arguments with additional
metadata or behaviour. Use `@Directive(sdl)` to attach schema-level directives.

## Schema directives (SDL)

```ts
import { Directive, Field, ObjectType } from '@banhmi/graphql'

@ObjectType()
@Directive('@deprecated(reason: "Use NewCat instead")')
export class Cat {
  @Field()
  @Directive('@deprecated')
  legacyName: string
}
```

## Custom directive transformer

Register a schema transformer in `GraphQLModule`:

```ts
GraphQLModule.forRoot({
  autoSchemaFile: true,
  transformSchema: (schema) => applyMyDirective(schema),
})
```

## Apollo Federation directives

See [Federation](/graphql/federation) for `@key`, `@external`, `@extends`.

## API reference

- `@Directive(sdlString)` — attach an SDL directive to a type, field, or argument.
- `transformSchema` option — apply `graphql-tools` schema transformers.
