# Complexity

Query complexity analysis prevents clients from sending deeply nested or
expensive queries that could overload the server.

## Setup

```bash
bun add graphql-query-complexity
```

```ts
import { GraphQLModule } from '@banhmi/graphql'
import { fieldExtensionsEstimator, simpleEstimator } from 'graphql-query-complexity'

GraphQLModule.forRoot({
  autoSchemaFile: true,
  plugins: [
    {
      requestDidStart: () => ({
        didResolveOperation({ request, document }) {
          const complexity = getComplexity({
            schema,
            query: document,
            variables: request.variables,
            estimators: [
              fieldExtensionsEstimator(),
              simpleEstimator({ defaultComplexity: 1 }),
            ],
          })
          if (complexity > 100) {
            throw new Error(`Query complexity ${complexity} exceeds limit of 100`)
          }
        },
      }),
    },
  ],
})
```

## Per-field complexity

```ts
import { Extensions, Field, ObjectType } from '@banhmi/graphql'

@ObjectType()
export class Cat {
  @Field(() => [Toy])
  @Extensions({ complexity: ({ childComplexity }) => 10 + childComplexity })
  toys: Toy[]
}
```

## API reference

- `graphql-query-complexity` — third-party library; install separately.
- `fieldExtensionsEstimator()` — reads complexity from `@Extensions`.
- `simpleEstimator({ defaultComplexity })` — flat cost per field.
- `getComplexity({ schema, query, variables, estimators })` — compute total cost.
