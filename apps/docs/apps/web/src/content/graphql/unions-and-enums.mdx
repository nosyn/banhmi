# Unions and Enums

## Unions

Unions represent a field that can return one of several object types.

```ts
import { createUnionType, ObjectType } from '@banhmi/graphql'

@ObjectType() class Cat { /* ... */ }
@ObjectType() class Dog { /* ... */ }

export const PetUnion = createUnionType({
  name: 'Pet',
  types: () => [Cat, Dog],
  resolveType(value) {
    return value.indoor !== undefined ? 'Cat' : 'Dog'
  },
})
```

Use in a resolver:

```ts
@Query(() => [PetUnion])
pets() {
  return this.petsService.findAll()
}
```

## Enums

```ts
import { registerEnumType } from '@banhmi/graphql'

export enum PetStatus {
  AVAILABLE = 'AVAILABLE',
  ADOPTED = 'ADOPTED',
}

registerEnumType(PetStatus, { name: 'PetStatus' })
```

Use as a field type:

```ts
import { Field, ObjectType } from '@banhmi/graphql'

@ObjectType()
export class Cat {
  @Field(() => PetStatus)
  status: PetStatus
}
```

## API reference

- `createUnionType({ name, types, resolveType })` — define a union type.
- `registerEnumType(enum, { name, description? })` — register a TypeScript enum as a GraphQL enum.
