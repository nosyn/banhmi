# HTTPS & Multiple Servers

**How do I enable HTTPS?**

Pass TLS config to `BanhmiFactory.create`:

```ts
const app = await BanhmiFactory.create(AppModule, {
  https: { key: pemKey, cert: pemCert },
})
await app.listen(443)
```

**Can I run HTTP and HTTPS simultaneously?**

Create two separate application instances from the same module:

```ts
const http  = await BanhmiFactory.create(AppModule)
const https = await BanhmiFactory.create(AppModule, {
  https: { key, cert },
})

await http.listen(80)
await https.listen(443)
```

Each instance has its own `BunAdapter` and `Bun.serve` process.

**Where do I get a TLS certificate for local dev?**

```bash
openssl req -x509 -newkey rsa:2048 -keyout server.key -out server.cert \
  -days 365 -nodes -subj '/CN=localhost'
```
