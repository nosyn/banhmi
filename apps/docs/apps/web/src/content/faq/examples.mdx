# Examples

The `examples/` directory in the repository contains runnable sample
applications:

| Directory | Description |
|-----------|-------------|
| `examples/cats-api` | Classic CRUD REST API |
| `examples/better-auth-api` | Better Auth integration |
| `examples/graphql-demo` | GraphQL with subscriptions |
| `examples/microservices-demo` | Microservices with TCP transport |
| `examples/drizzle-api` | Drizzle ORM + SQLite |
| `examples/edge-worker` | Cloudflare Workers / Vercel Edge via `@banhmi/edge` |
| `examples/lambda-app` | AWS Lambda via `@banhmi/serverless` |

## Running an example

```bash
# From the repo root
cd examples/cats-api
bun install
bun run dev   # or: bun src/main.ts
```

## Edge worker example

```bash
cd examples/edge-worker
bun run dev          # starts Bun.serve wrapping the edge handler
bun test             # runs integration tests
```

## Lambda app example

```bash
cd examples/lambda-app
bun test             # invokes handler with synthetic API Gateway events
```
