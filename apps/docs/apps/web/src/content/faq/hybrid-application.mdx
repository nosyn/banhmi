# Hybrid Application

**Can I run HTTP and a microservice transport in the same process?**

Yes — use `connectMicroservice` and `startAllMicroservices`:

```ts
import { BanhmiFactory } from '@banhmi/platform-bun'
import { tcpTransport } from '@banhmi/microservices'
import { AppModule } from './app.module'

const app = await BanhmiFactory.create(AppModule)
app.connectMicroservice({ transport: tcpTransport({ port: 3001 }) })
await app.startAllMicroservices()
await app.listen(3000)
```

**In what order should I call the APIs?**

1. `BanhmiFactory.create(AppModule)` — build the DI graph.
2. `app.connectMicroservice(opts)` — register transports (repeatable).
3. `app.startAllMicroservices()` — start all transports concurrently.
4. `app.listen(port)` — open the HTTP server.

**Can I register multiple transports?**

Yes — call `connectMicroservice` once per transport.  All registered
transports are started concurrently by `startAllMicroservices`.
