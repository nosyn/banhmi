# Raw Body

**How do I access the raw request bytes for webhook verification?**

Enable `rawBody` when creating the application:

```ts
const app = await BanhmiFactory.create(AppModule, { rawBody: true })
```

Then read `ctx.rawBody` in any handler:

```ts
@Post('/webhook')
async receive(ctx: BunRouteCtx) {
  const bytes = ctx.rawBody  // Uint8Array | undefined
  // compute HMAC, verify signature, etc.
}
```

**Why is `ctx.rawBody` undefined?**

The `rawBody: true` option was not passed to `BanhmiFactory.create`, or the
handler is calling `ctx.json()` / `ctx.text()` before checking `rawBody`.
Enable the option at the factory level.

**Does enabling `rawBody` affect performance?**

Yes — the request body is read once into a `Uint8Array` on every request.
For routes that don't need the raw bytes, consider splitting the app into a
separate instance without `rawBody: true`.
