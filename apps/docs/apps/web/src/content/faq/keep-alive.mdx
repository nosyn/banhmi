# Keep-Alive Connections

**Does Banhmi support keep-alive connections?**

Yes. `Bun.serve` enables keep-alive by default.  Banhmi exposes the
`idleTimeout` factory option to control how long idle connections stay open:

```ts
const app = await BanhmiFactory.create(AppModule, { idleTimeout: 30 })
await app.listen(3000)
```

**What is the default idle timeout?**

The default is whatever `Bun.serve` uses internally.  If you need explicit
control, set `idleTimeout` (in seconds).  Setting `0` disables the timeout.

**Can clients reuse connections across requests?**

Yes. HTTP/1.1 keep-alive is handled transparently by Bun.  No Banhmi
configuration is needed beyond the default.
