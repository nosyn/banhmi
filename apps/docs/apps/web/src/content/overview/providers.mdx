# Providers

Providers are the primary building blocks for business logic. Any class decorated
with `@Injectable()` can be injected into a controller or another provider through
Banhmi's static-`inject` DI container.

## When to use

- Encapsulate business logic that controllers should delegate to.
- Share stateful or stateless services across multiple controllers and modules.
- Wrap third-party clients (databases, HTTP clients, etc.).

## Setup

```ts
import { Module } from 'banhmi'
import { CatsController } from './cats.controller'
import { CatsService } from './cats.service'

@Module({
  controllers: [CatsController],
  providers: [CatsService],
})
export class CatsModule {}
```

## Usage

```ts
import { Injectable } from 'banhmi'

@Injectable()
export class CatsService {
  private readonly cats: Cat[] = []

  findAll(): Cat[] {
    return this.cats
  }

  create(dto: CreateCatDto): Cat {
    const cat: Cat = { id: crypto.randomUUID(), ...dto }
    this.cats.push(cat)
    return cat
  }
}
```

Inject into a controller using `static inject`:

```ts
@Controller('cats')
export class CatsController {
  static inject = [CatsService] as const
  constructor(private readonly catsService: CatsService) {}
}
```

## Value providers

```ts
@Module({
  providers: [{ provide: APP_NAME_TOKEN, useValue: 'My App' }],
})
export class AppModule {}
```

## Factory providers

```ts
@Module({
  providers: [
    {
      provide: DATABASE_TOKEN,
      useFactory: (config: ConfigService) => new Database(config.get('DB_URL')),
      inject: [ConfigService],
    },
  ],
})
export class AppModule {}
```

## Tokens

Use `Token<T>('NAME')` to create typed DI tokens:

```ts
import { Token } from 'banhmi'

export const DATABASE_TOKEN = Token<Database>('DATABASE_TOKEN')
```

## API reference

- `@Injectable()` — marks a class as a DI-managed provider.
- `static inject = [Dep, TOKEN] as const` — declare constructor dependencies.
- `Token<T>(name)` — create a typed injection token.
- `useValue` — bind a static value to a token.
- `useFactory` — bind the result of a factory function.
- `useClass` — substitute a different class implementation.
